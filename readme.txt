== Description ==

Automatically scans for emails and hides them from spammers using the reCAPTCHA Mailhide API. Nothing in the database is
actually modified, it's just modified once it's been taken out of the database. See it in action [here](http://www.jorgepena.be/wp-mailhide/).

== Requirements ==

You must have two Mailhide API keys (Private and Public). Get them [here](http://recaptcha.net/apidocs/mailhide/).
You must also have the [mcrypt](http://php.net/mcrypt) PHP module, most installations should have it.

== Installation ==

1. Read [this](http://www.jorgepena.be/wp-mailhide/) for a thorough guide.
